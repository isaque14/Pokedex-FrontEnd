Sobre o Projeto

O RotomDex faz parte de um conjunto de três projetos que foram desenvolvidos para a avaliação final da Academia de .NET 2023, que é um programa de formação de Desenvolvedores de software da Atos (Atos) ministrado pelos professores da Universidade Franciscana (UFN) (UFN), nesse link você pode vizualisar o gitHub da Academia .NET 2023 e encontrar todo o conteúdo que foi ministrado (Academia .NET) Para este projeto final, foram desenvolvidas 3 aplicações (Front-End, Back-End e um Bot), o Bot foi feito para monitorar os outros dois projetos (Front-End e Back-End) e também os serviços utilizados pelo back-end, caso algo não esteja funcionando corretamente com os serviços ou projetos, um email é disparado pelo bot informando o administrador do sistema.
